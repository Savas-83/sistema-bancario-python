# Sistema Bancário em Python (Versão com Funções)

## Objetivo

Implementar um sistema bancário simples no terminal, utilizando funções para organizar o código e aplicar regras específicas de argumentos:

- `depositar()` → argumentos apenas por posição
- `sacar()` → argumentos apenas por nome
- `mostrar_extrato()` → combina posição e nome

Além disso, foram adicionadas novas funcionalidades:

- Cadastro de usuário (com validação de CPF)
- Criação de conta bancária
- Lista de contas criadas


## Funcionalidades

✅ Depositar  
✅ Sacar com limite e controle de quantidade  
✅ Extrato bancário  
✅ Criar usuário  
✅ Criar conta vinculada ao usuário  
✅ Listar contas
